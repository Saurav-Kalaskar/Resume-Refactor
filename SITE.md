# ATS Resume Refactoring Engine

## Vision
A single-page React app dashboard with a premium macOS native window feel.

## Sitemap
- [ ] App (Main Dashboard)

## Roadmap
1. Build the App dashboard with glassmorphism UI.

## Components Needed
1. **Header:** macOS window toolbar with the traffic light buttons.
2. **Job Description Text Area:** A premium text input area for users to paste a job description.
3. **Drag-and-Drop File Upload Zone:** A distinct, glowing drop zone for uploading a base resume.
4. **'Company Name' Input:** A sleek input field.
5. **Action Area:** A prominent submit/refactor button with glowing hover effects.
6. **Responsive PDF Preview Container:** A split-pane or dedicated container to show the generated PDF, with a mobile fallback view.
